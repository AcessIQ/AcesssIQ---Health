# AccessIQ — Medicaid Impact Calculator (Static Site)

This folder is ready to upload to **GitHub Pages** (free, no card).

## Quick Deploy (GitHub Pages)
1) Go to https://github.com → **New repository** → name it e.g., `accessiq-calculator` → set **Public** → **Create**.
2) Click **Add file → Upload files** → upload everything in this folder (**index.html** and this README).
3) Go to **Settings → Pages**:
   - **Source:** `Deploy from a branch`
   - **Branch:** `main` (or `master`) and `/ (root)` → **Save**
4) Wait ~30–60 seconds. Your site will be live at:
   `https://<your-username>.github.io/accessiq-calculator/`

### Notes
- `index.html` is your entry point.
- If you add more assets later (images, CSS/JS), keep **relative paths** and upload them too.
- Hard refresh to bust cache: **Cmd/Ctrl+Shift+R**.
